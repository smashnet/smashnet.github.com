Released Version 0.05

Mon, Dec 17 2012, 14:42

Together with the new website I release the new version 0.05 of PG. There are quite some features you might have been waiting for:

Change Log:

  • Encrypted data transfers (using AES-128) (AES-256 will come in 0.06)
  • Settins are now stored in config.xml (no need for long start script anymore)
  • XMPP password is no longer shown in process list
  • Warns you to manually forward the P2P port in the firewall if no UPnP capable router was found
  • Fixed a bug that prevented you from joining a password protected channel
  • Fixed a bug that caused directories to be not deleted properly

Have fun with this new release! Looking forward to your comments in the discussion forum!

Cheers,
Nico

Welcome to pgrp.de

Mon, Dec 17 2012, 13:50

I'm glad you found your way to the brand new website of the P2P syncing tool Peergroup. I previously used to publish news about Peergroup in it's GitHub Wiki and on my other blog smashnet.de. Unfortunately there was no single place where you could get all information and discuss about the project, so this little website arose.

All new versions of Peergroup will be commented here with a blog post, telling you which new features were implemented and anything else what seems important to me :)

Have a look at the top-right of the site! I am very happy about every participation in the discussion group. Every idea, hint or whatever is welcome!

Cheers,
