In this section you can download two different versions of Peergroup. The main version and the development version. Deciding between these versions is quite easy:

If you want a more stable experience, use the main version.
If you want to test the newest functions, use the development version.

Keep in mind that different versions of Peergroup should not be mixed. It can work, but it is not supported!

Requirements: MacOS/Linux/Windows running Java 7 or newer.
